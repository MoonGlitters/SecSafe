Principales Amenazas Cibernéticas

Phishing
¿Qué es?

Técnica de engaño que suplanta identidades para robar datos sensibles mediante correos, SMS o páginas web falsas.

Cómo Protegerse:
  • Verifica la URL y el remitente
  • No descargues archivos sospechosos
  • Usa autenticación en dos pasos (2FA)
Saber más
Ransomware
¿Qué es?

Malware que encripta archivos y exige un rescate (generalmente en criptomonedas) para liberarlos.

Cómo Protegerse:
  • Haz copias de seguridad periódicas
  • Actualiza sistemas y software
  • No pagues el rescate (no garantiza recuperar datos)
Saber más
Ataque DDoS
¿Qué es?

Inunda servidores o redes con tráfico falso para saturarlos y dejarlos inoperativos.

Cómo Protegerse:
  • Usa servicios de mitigación DDoS (ej. Cloudflare)
  • Configura firewalls y límites de tasa de solicitudes
Saber más
Malware
¿Qué es?

Software malicioso que infecta dispositivos para robar datos, espiar o dañar sistemas. Incluye virus, troyanos y spyware.

Cómo Protegerse:
  • Usa antivirus actualizado
  • Descarga software solo de fuentes oficiales
